🎓 Student Management System

A professional C++ console application for managing student data with full CRUD operations and file handling.

---

 🚀 Features

- ➕ Add Student
- 📋 Display Students
- 🔍 Search Student by ID
- ❌ Delete Student
- ✏️ Update Student
- 💾 Persistent storage using file handling

---

 🛠 Technologies Used

- C++
- File Handling (fstream)
- Data Structures (Vector)

---

 📂 How It Works

- All student data is stored in a file (`students.txt`)
- Data is automatically loaded when the program starts
- Changes are saved instantly after operations
